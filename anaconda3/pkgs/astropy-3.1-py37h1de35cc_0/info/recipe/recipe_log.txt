    Properly specify Numpy version dependencies.
    
    To maximize binary compatibility, we should *build* against as low of a
    version of Numpy as we can get away with, and then specify that we can *run*
    against any later Numpy version. The lowest versions in conda-forge are 1.7,
    1.9, and 1.11 for Python 2.7, 3.5, and 3.6 respectively. But Astropy requires
    1.10, so we enforce that minimum version for the older Python releases.
